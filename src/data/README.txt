The experiments can be re-run by executing the 'run.sh' shell script. This can be done by executing 'sh run.sh' on the command line.

The data from the experiments can be found in csv files in the 'data' directory. 

'data/metric1.csv' - refers to the data retrieved using the first metric.
'data/metric2.csv' - refers to the data retrieves using the second metric. 
'data/both_averages.csv' - refers to the two averages of the data from the first and second metric.
